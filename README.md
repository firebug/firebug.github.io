#Firebug Central Docs

##Contribute

Clone the Repo:

```
git clone https://github.com/firebug/firebug.github.io.git
```

Change into the directory

```
cd firebug.github.io
```
Retrieve the dependencies using

```
bundle install
```
build it

```
bundle exec jekyll serve
```
